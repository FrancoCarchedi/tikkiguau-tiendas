# Objetivo

El objetivo es analizar las dependencias que se utilizarán en este proyecto. Luego proceder con la instalación y configuración de cada una. Solo hacer la configuración necesaria, completa y lista para usar. No hace falta desarrollar ni implementar casos de prueba o ejemplos para probarlo, posteriormente lo haré yo.
Por favor, generar un archivo DOCUMENTACION.md donde haya una guía sobre cada dependencia y como usar cada recurso.
